This project demonstrates several concepts used in developer tooling and complex UI systems:

• Node-based editor architecture similar to visual scripting tools
• MVVM-based UI architecture in WPF
• Graph serialization and deserialization
• Dynamic node creation and connection management
• Extensible UI components for complex workflows

## Architecture

The application follows MVVM architecture to separate UI logic from application logic.
